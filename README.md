<p class=MsoNormal><b style='mso-bidi-font-weight:normal'><span lang=EN>Primary
capsule layer<o:p></o:p></span></b></p>

<p class=MsoNormal><span lang=EN><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span lang=EN>It consists of 32 6x6x8 capsules that
translates to 32 convolution layers each have 8 filters and 9x9 kernels, stride
of 2 and linear activation. The output (<b style='mso-bidi-font-weight:normal'>u</b>)
is 32x6x6x8 and is reshaped to 1152x8.<span style='mso-spacerun:yes'> 
</span>1152 is total capsules outputs.<span style='mso-spacerun:yes'>  </span></span></p>

<p class=MsoNormal><b style='mso-bidi-font-weight:normal'><span lang=EN>Routing<o:p></o:p></span></b></p>

<p class=MsoNormal><b style='mso-bidi-font-weight:normal'><span lang=EN><o:p>&nbsp;</o:p></span></b></p>

<p class=MsoNormal><span lang=EN>At this stage, the logits (<b
style='mso-bidi-font-weight:normal'>bij</b>) will be learned through routing
algorithms.<span style='mso-spacerun:yes'>  </span>Logits will be translated to
coupling coefficients (<b style='mso-bidi-font-weight:normal'>cij</b>) using
softmax function (calculated over DigitCaps). This defines the parent (Digit
Caps 1 to 10) that is chosen by each capsule output. So the output is called (<b
style='mso-bidi-font-weight:normal'>s</b>). (s) then goes through squash
function to create (<b style='mso-bidi-font-weight:normal'>v</b>) with unit
norm. <span style='mso-spacerun:yes'> </span>Routing algorithm should be
executed for each sample in the batch, so I used <b style='mso-bidi-font-weight:
normal'>tf.while_loop.<o:p></o:p></b></span></p>
